services

A house is only as strong as its foundation.

Beautiful HTML is the foundation of a beautiful website. Good CSS and responsive JavaScript can only exist with equally good markup. I build clean, semantic and accesible code.

CSS & SaSS

Using the SaSS pre-processor, I generate readable, consistantly formatted CSS in less time. Used in combination with Bootstrap v5, I can build customizable templates, resulting in reduced production time. Incorporating SaSS enables the creation of component based style sheets, allowing for quick future modification.

JavaScript

If clients just wanted to read, they would buy a book. JavaScript allows for dynamic features and engagement on your website. Utilizing DOM manipulation and REST API integration, builds will deliver relevent information and keep clients engaged longer.

mobile responsive

Globally, 68.1% of all website visits in 2020 came from mobile devices. Creating content that visually scales from one platform to the next is paramount when trying to get your message across. I create a product that is engaging on all device sizes.
